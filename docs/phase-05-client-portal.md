# Fase 05: Portal cliente

## Alcance implementado

- Dashboard cliente con métricas reales
- Listado de tickets por clientId
- Creación de ticket desde cliente
- Detalle de ticket desde cliente
- Respuesta del cliente en la conversación
- Navegación dedicada para cliente

## Seguridad

- El cliente solo ve tickets de su propio clientId
- El cliente crea tickets ligados a su propio clientId
- El cliente solo puede responder en tickets de su propio clientId
